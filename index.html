Projects

Projects mentioned here were made out of my own curiosity.

CodinCloud [WIP] is a compiler as a service platform prototype written using python flask framework as the backend service and MaterializeCSS on the frontend. The idea is to have a platform similar to cloud9 ide. A demo of the site is available at heroku. Github

Meltdown [WIP] is a download manager written in Rust. Splits the task given to it in fixed sized chunks and downloads them in parallel. Uses Rust’s message passing Channels api for inter-thread communication.
As of now has a rudimentary UI integration using gtk-rs framework and does not have a resume support, which i intend to integrate in next alpha release when time permits. Github

Silica [WIP] is a static site generator written in Rust. The site uses the doc-opt command line argument parser in rust and supports writing content in markdown and renders them to html. A custom template engine is in progress for next alpha release. Github
